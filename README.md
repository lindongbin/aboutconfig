[UC Manager](UserChromeRules.uc.js) is a userChrome CSS and JS management tool entirely generated by AI, providing a visual rule management interface.

## Caution

- Recommended to be used in single-window mode only
- Does not load CSS and JS files from the chrome folder
- Requires manual addition or import of CSS and JS code
- All rules are centrally stored in a single JSON file

## Installation

1. Copy [config-prefs.js](install_folder/defaults/pref/config-prefs.js) to Firefox installation directory at `defaults/pref/config-prefs.js`
2. Copy [UserChromeRules.uc.js](UserChromeRules.uc.js) to Firefox installation directory, in the same directory as `firefox.exe`
